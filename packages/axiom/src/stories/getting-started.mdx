# Axiom Design System

Axiom is built on top of Mantine and provides a set of reusable UI components tailored for our applications.

<MantineProvider>
  <Banner color="yellow" title="Under active development." mb="xl">
    <span>Several legacy components are pending migration as we evaluate the most effective integration strategies with Mantine's architecture. Specifically, we are currently analyzing native Mantine patterns for handling modals, notifications, and form state management to ensure optimal performance and DX before standardizing these components.</span>
  </Banner>
  <Banner color="blue" title="Note" mb="xl">
    <em>Component APIs, names, and structural patterns are subject to change without notice during this transitional phase.</em>
  </Banner>
</MantineProvider>
