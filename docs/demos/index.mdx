## Running the demos locally

All demos are available in Storybook:

```bash
pnpm storybook
```

Navigate to the **Demos** section in the Storybook sidebar to interact with each
one, or open the walkthrough pages above to see them embedded alongside an
explanation of the patterns they use.

## Enterprise-grade quality

These demos represent patterns validated in production at enterprise scale,
serving thousands of users daily. Every component shown here is built for:

- **Accessibility** — WCAG 2.1 AA compliance and full keyboard operability.
- **Performance** — optimized rendering with minimal re-renders.
- **Responsiveness** — fluid layouts that adapt to every screen size.
- **Theming** — complete light/dark mode support out of the box.
