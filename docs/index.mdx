## What makes Axiom different

| Principle | What it means |
|-----------|---------------|
| **Token-first** | Every color, spacing, and sizing value comes from a token — no magic numbers. |
| **Mantine-powered** | Built on Mantine v9 so you get a battle-tested base with Axiom opinions on top. |
| **Themeable** | One theme object controls the entire look — swap it and everything updates. |
| **Accessible** | WCAG AA compliance baked in. Keyboard, screen reader, and high-contrast support by default. |

## Architecture at a glance

```
axiom-ui
├── ThemeProvider      → Wraps your app, provides tokens + theme
├── Components         → 30+ ready-to-use UI building blocks
└── Foundations        → Colors, typography, spacing, icons
```

Every component is a thin wrapper around a Mantine primitive. Axiom controls the defaults — you control the overrides.
